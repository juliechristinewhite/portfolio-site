PSD Editing

PSD Files are well organized layer name
All elements are 100% scalable vector graphics and you can free scale, distort, grow, reduce size and edit.

You can easy to change the color style of text, icons and placeholders:

  1. Double click on thumbnail of selected layer in Layers panel
  2. Click on the Color Picker in the tool bar
  3. Change the color

Edit Images / Smart Objects (Logo, Photos, Sliders etc...)

You can easy to replace the images with Smart Objects. Follow the 4 simple steps:

Step-1: Copy your image from photoshop
Step-2: Go to Template file, open LAYER PANEL from "Window" menu of Photoshop (shotcut:F7), Select Layer of photo (you can find Layer name "PHOTO - Double Click Here"), Double Click on Thumbnail of that Layer
Step-3: When you double click, you will get a new file open, Pest your image here.
Step-4: Save that new open file and close... It's DONE :)
